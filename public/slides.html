## GitHub - Open Source collaboration - JavaScript is most used language here, with 21% (feb 2013) - Popular starred (feb 2013): bootstrap, _node,_ jquery, html5-boilerplate, _rails,_ impress.js, d3, backbone, font-awesome, _homebrew_
## Travis CI - Continous Integration server for open source projects - GitHub integration - Simple config, - 4 line config file to run tests for node.js project - Can test HTML5 in headless browser ---- ## Testling CI - Cross-browser testing
### C/C++ • LLVM • Emscripten - C-Libraries such as libSDL implemented on top of HTML5 (canvas, etc.) - Ported app examples: LaTeX, zlib, OpenTTD, Python, Lua, Poppler, SuperTux, FreeType, SQLite, gnuplot, Cube 2: Sauerbraten, opendune, Qt, ... - Useful for porting existing C/C++ code to run in JavaScript - Just install llvm/clang-3.2 and checkout the git repository - Performance cost
## Java • GWT - Write JavaScript via Java - Build on Java: developers, project-scalability, static typing, tooling, ... - New Java-APIs for web-applications
